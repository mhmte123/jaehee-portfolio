Overview

프로젝트 소개

마뗑킴(Matin Kim)은 국내 패션 브랜드로 마뗑킴과 마틴킴 두 라인을 운영하고 있습니다. 기존 쇼핑몰의 사용자 경험을 분석한 결과, 브랜드 전환 구조가 직관적이지 않고 상품 탐색 및 구매 흐름에서 여러 문제점이 발견되었습니다.

4인 팀 프로젝트로 진행하였으며, 저는 메인 페이지·상품 리스트·상세페이지·장바구니·주문 페이지의 UI 퍼블리싱 및 인터랙션 구현을 주도했습니다. HTML5 시맨틱 마크업, CSS3, JavaScript, React, Next.js를 활용했으며 Git 기반 협업으로 진행했습니다.

Problem

발견한 문제점

UX 문제

브랜드(마뗑킴/마틴킴) 전환 구조가 직관적이지 않아 탐색성이 낮음
헤더 내 카테고리 구분이 명확하지 않아 메뉴 접근성이 떨어짐
검색 기능 외 상품 탐색 수단이 부족하여 원하는 상품 탐색이 어려움

UI 문제

상품 리스트의 과도한 썸네일 크기와 불균형한 간격으로 정보 밀도가 낮음
상세 페이지의 대형 이미지 중심 구조로 인해 스크롤 피로도가 높음
장바구니 → 주문 → 결제 단계별 흐름이 불명확

Solution

해결 방법

브랜드 전환 상태를 직관적으로 인식할 수 있도록 히어로 섹션 내 토글 UI 설계
Grid 기반 상품 리스트 재구성으로 정보 밀도 및 가독성 향상
필터 UI를 추가하여 카테고리별 상품 탐색 편의성 개선
상품 상세 정보와 이미지를 재배치하여 스크롤 이동 거리 최소화
장바구니 → 주문 → 결제 단계별 흐름을 고려한 화면 구조 개선
시맨틱 마크업 적용 및 CSS 구조 정리로 유지보수성 향상
AI(OpenAI Codex)를 활용한 HTML 구조 개선 및 코드 리팩토링

Result

성과

브랜드 접근성

토글 UI로 브랜드 전환 인지성 향상

상품 탐색

필터 기능 추가로 상품 접근성 개선

레이아웃 최적화

한 화면 내 노출 상품 수 증가

코드 품질

시맨틱 마크업 및 리팩토링으로 유지보수성 향상
