Overview

프로젝트 소개

Netflix 웹사이트의 사용자 경험과 시각적 구조를 재설계한 리뉴얼 프로젝트입니다. 기존 서비스의 UI를 면밀히 분석하여 콘텐츠 탐색의 흐름을 개선하고, 브랜드 정체성을 유지하면서도 보다 직관적이고 세련된 인터페이스를 구현했습니다.

Next.js 기반으로 직접 개발하여 Vercel에 배포하였으며, 디자인 의도가 실제 서비스에 그대로 구현되는 과정을 주도했습니다.

Problem

발견한 문제점

UX 문제

홈 화면의 콘텐츠 밀도가 높아 원하는 콘텐츠를 찾는 탐색 시간이 길어짐
카테고리 간 이동 구조가 직관적이지 않아 탐색 흐름이 끊김
콘텐츠 정보(장르, 줄거리 등) 확인을 위한 인터랙션이 번거로움

UI 문제

히어로 배너의 시각적 위계가 약해 핵심 콘텐츠 강조가 부족
썸네일 레이아웃이 단조로워 콘텐츠 간 차별화가 어려움
반응형 처리가 미흡하여 다양한 화면 크기에서 레이아웃이 무너짐

Solution

설계 방향

UX 전략

히어로 섹션 개편으로 핵심 콘텐츠의 시각적 집중도 강화
카테고리 네비게이션 구조 재설계로 탐색 흐름 단순화
호버 인터랙션으로 콘텐츠 정보를 자연스럽게 노출

UI/개발

Netflix 브랜드 컬러·타이포그래피를 유지하며 레이아웃 밀도 최적화
Next.js App Router 기반 페이지 구조 설계 및 구현
반응형 레이아웃으로 모바일·태블릿·데스크탑 모두 대응

Result

성과

탐색 경험

콘텐츠 카테고리 구조 개선으로 탐색 흐름 단순화

기술 성과

Next.js로 풀스택 구현 및 Vercel 배포 완료

UI 완성도

시각적 위계와 레이아웃 밀도 최적화로 브랜드 일관성 향상

반응형

전 디바이스 대응 레이아웃 구현
